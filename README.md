# Starfall Sprint (Processing)

A Java game built in the Processing environment by recreating a browser game I made using Codex from scratch.

This project is part of my journey to strengthen my object-oriented programming, software engineering, and game development skills before starting Computer Science at the University of Waterloo.

---

## Project Goals

The purpose of this project is to:

- Practice object-oriented programming (OOP)
- Improve Java programming skills
- Learn professional Git & GitHub workflows
- Build a game from scratch without copying implementation code
- Develop better software architecture through iterative refactoring

---

## Current Features

- Player class
- Object-oriented project structure
- Triangle player
- Continuous WASD movement
- Screen boundary detection
